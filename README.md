## Installation

To install:

> [p]repo add wdk-cogs <https://github.com/bogdanwdk/red-cogs>  
> [p]cog list wdk-cogs  
> [p]cog install wdk-cogs &lt;cog&gt;  
> [p]load &lt;cog&gt;  
> [p]help &lt;Cog&gt;
